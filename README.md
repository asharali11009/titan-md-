    Clone the repository

Bash

git clone https://github.com/ezio/titan-bot.git
cd titan-bot

Install dependencies

Bash

npm install

Configuration

    Edit the settings.js file to set your name and phone number
    Customize any other settings as needed

Launch the bot

Bash

    npm start

    Authentication
        Scan the QR code with WhatsApp (Linked Devices section)
